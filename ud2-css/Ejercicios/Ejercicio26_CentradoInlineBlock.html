Ejercicio Práctico: Centrar una Fila de Chips

A continuación, tenemos una serie de elementos "Chip" que deberían estar centrados horizontalmente, incluso cuando el ancho de la pantalla obliga a que salten de línea.
¡Observa el comportamiento de `inline-block` y ajusta la ventana para ver cómo se centra cada fila!

El contenedor padre usa text-align: center;

CSS Básico
HTML5
Modelo de Cajas
Flexbox
Grid CSS
JavaScript ES6
ReactJS
Tailwind CSS
SASS
Accesibilidad

Tarea:

1. Inspecciona el código CSS del **`.contenedor-chips`** (el padre).

2. Inspecciona el código CSS del **`.chip`** (los hijos).

3. **Pregunta de Auto-Evaluación:** Si cambiaras el `display` del `.chip` a `block`, ¿qué ocurriría? ¿Y si lo cambiaras a `inline`?
